# notes-electron-app

<h1>MOTES</h1>
<p>A simple and minimalistic note taking app using markdown built using ElectronJS </p>


## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
